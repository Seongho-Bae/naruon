# Issue 76: Refactor Configs to Tenant-based DB Storage Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Remove tenant-specific configuration (SMTP, IMAP, POP3, OpenAI, Google Calendar API keys) from the global environment (`core/config.py`) and store them per-user (tenant) in the database.

**Architecture:** Create a `TenantConfig` model in the database. Update all service functions (`email_client.py`, `llm_service.py`, `calendar_service.py`, `embedding.py`, `imap_worker.py`) to accept a `user_id` or `tenant_config` parameter rather than reading from `settings`. Create an API endpoint (`/api/config`) for users to set these values.

**Tech Stack:** Python, FastAPI, SQLAlchemy, PostgreSQL.

Modify `backend/db/models.py` to add `TenantConfig`:
```python
class TenantConfig(Base):
    __tablename__ = "tenant_configs"
    
    id: Mapped[int] = mapped_column(primary_key=True)
    user_id: Mapped[str] = mapped_column(String, unique=True, index=True)
    
    # Email settings
    smtp_server: Mapped[str] = mapped_column(String, nullable=True)
    smtp_port: Mapped[int] = mapped_column(nullable=True)
    smtp_username: Mapped[str] = mapped_column(String, nullable=True)
    imap_server: Mapped[str] = mapped_column(String, nullable=True)
    imap_port: Mapped[int] = mapped_column(nullable=True)
    pop3_server: Mapped[str] = mapped_column(String, nullable=True)
    pop3_port: Mapped[int] = mapped_column(nullable=True)
    
    # OAuth and Third Party Settings
    oauth_client_id: Mapped[str] = mapped_column(String, nullable=True)
    oauth_client_secret: Mapped[str] = mapped_column(String, nullable=True)
    oauth_redirect_uri: Mapped[str] = mapped_column(String, nullable=True)
    
    openai_api_key: Mapped[str] = mapped_column(String, nullable=True)
    google_client_id: Mapped[str] = mapped_column(String, nullable=True)
    google_client_secret: Mapped[str] = mapped_column(String, nullable=True)
```

Modify `backend/core/config.py` to REMOVE the tenant-specific configs. Keep only `DATABASE_URL`, `DEBUG`, `OPENAI_EMBEDDING_MODEL`, `OPENAI_MODEL` (since models are global system choices).

Modify `backend/tests/test_config.py` to REMOVE checks for the removed configs.
